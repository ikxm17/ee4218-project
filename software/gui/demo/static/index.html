Note on HDL inference wall-clock: the HDL "inference" value includes the AXI-Lite pixel preload (~150 ms for 65 K pixel writes). The accelerator's active compute is reported separately as cycles → time at 100 MHz.

FPS columns: compute is the throughput of the core inference engine alone — 1000 / inference_ms for TFLite (the invoke() wall-clock) and 1000 / cycle_time_ms for the HDL (cycles at 100 MHz). This is the apples-to-apples accelerator-vs-CPU comparison. pipeline is 1000 / total_ms — end-to-end throughput including preprocess, bus transfers, and postprocess, i.e. what you would observe in a real streaming loop.
